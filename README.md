# Awesome Chocolates - Exploratory Data Analysis

## Overview

This project demonstrates an end-to-end **Exploratory Data Analysis (EDA)** using the **Awesome Chocolates** dataset across multiple data analytics tools.

The goal was to perform the same analysis using different technologies to better understand their strengths and how they fit into a real-world analytics workflow.

## Technologies Used

* Microsoft SQL Server (T-SQL)
* Python (Jupyter Notebook)
* Pandas
* Matplotlib
* Seaborn
* Power BI

## Project Workflow

* Database exploration
* Schema understanding
* Data quality checks
* Missing value and duplicate detection
* Descriptive statistics
* Sales performance analysis
* Customer analysis
* Product analysis
* Geographic analysis
* Time-series analysis
* Dashboard creation in Power BI

## Key Skills Demonstrated

* Writing SQL queries for data exploration and business analysis
* Joining and transforming data from multiple tables
* Performing exploratory data analysis using Python
* Creating data visualizations with Matplotlib and Seaborn
* Building an interactive Power BI dashboard
* Applying the same analytical workflow across multiple tools

## Project Objective

Rather than learning each tool independently, this project focuses on solving the same business problem using different technologies to understand when and where each tool is most effective.
